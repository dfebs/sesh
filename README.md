# Sesh
A Ruby on Rails web app for tracking workouts.

## Requirements
Ruby Version: 3.2 or newer

## How to set up and run
1. `git clone https://github.com/dfebs/sesh.git`
1. `cd sesh`
1. `bundle install`
1. `bin/rails db:setup`
1. `bin/rails server`
